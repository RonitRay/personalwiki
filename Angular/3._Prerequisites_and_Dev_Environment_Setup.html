Created Saturday 10 July 2021

To work with Angular you need:

  1. HTML
  2. CSS
  3. TypeScript

To run Angular, you need:

  1. Node.js
+Why Node?
  1. NPM

[npm](https://www.npmjs.com/)

  1. Angular CLI
  2. A text editor (preferably VSCode)
  3. A browser (preferably Google Chrome)

NOW: First Angular app!

ng new firstAngular
cd firstAngular
ng serve
