KDE Setup Guide

Created Saturday 10 July 2021

NOTE: KDE is one of the best Linux DEs. It is super customizable and keeps getting better looking and faster thanks to an awesome dev team. It was my DE of choice for several years and I still really like it, although I use standalone (most often tiling) window-managers more nowadays for the extra functionality and ease of setup. I still recommend KDE to most people, but the truth is that there's quite some stuff I need to do every time I set up a fresh KDE environment to get it to where I find it mostly frictionless to use.

These notes are basically from around 2018 so they could be a little outdated, but they were steps for a completely new linux user to set up a new Kubuntu/KDE Neon install and get it close to Windows in terms of workflow and usability.

Before Install:

set BIOS boot mode to Legacy, not UEFI

During Install:

in the screen that asks how you want to install it. choose "Manual" or "something else" depending on what shows up. not "guided"

it will show you your drives. click on the main drive and click "New Partition Table"

Here are the partitions you need:

30-50GB primary partition

mount point /

filesystem ext4

(total size - RAM size - 2)GB primary partition

[i.e. if you have 900 GB remaining and 8gigs of RAM, put 900-8-2 or 890 GB for this partition]

mount point /home

filesystem ext4

(RAM size + 2)GB primary partition

type/mnt point swap

then proceed.

After Install:

Right click taskbar, switch to icons-only task manager

Right click taskbar, click unlock widgets if available else forget.

Right click taskbar, click panel options->add widget->minimize all windows. then drag it to extreme right of panel.

Right click taskbar, click panel options->lock widgets

Right click the clock, configure digital clock, click on "show date" if you like. I did.

Settings->Appearance->Workspace Theme->Look and Feel: Breeze Dark

Settings->Appearance->Workspace Theme->Desktop Theme: Breeze Dark

Settings->Desktop Behaviour-> Workspace-> Double-click to open files and folders

Settings->Desktop Behaviour-> Screen Edges-> Remove top-left corner

Settings->Window Management->Window Behaviour->Advanced-> Placement: Centered

Settings->Window Management->Task Switcher->Visualization: Thumbnails and uncheck show selected window

Settings->Input Devices->Mouse->Advanced->acceleration: 2.0x

>threshold 0

>dclick interval 400ms

>drag start time 500ms

>drag start dist 4px

>wheel scrolls by 3 lines

Settings->Input Devices->Touchpad->Scrolling->Reverse Scrolling: Vertical

Update procedure for KDE components:

Terminal (konsole):

sudo pkcon refresh

sudo pkcon update

Update procedure for everything else:

sudo apt update

sudo apt upgrade

Discover software center: search for Synaptic

Install Synaptic Package Manager

launch Synaptic

search for kubuntu-driver-manager

right click

mark for installation

select "mark" in the popup menu

click "Apply"

(follow this procedure for all future package installations via synaptic)

Settings->Driver Manager-> install whatever needed, then reboot

for office apps:

launch Synaptic

search for libreoffice

mark libreoffice and libreoffice-style-breeze from list

Apply

in synaptic search for

ttf-ms

mark ttf-mscorefonts-installer

apply

in the window that says installing software, click on details.click in the terminal window, press enter till given a prompt to accept eula, then type yes and press enter

launch LibreOffice Writer

Tools->Options->LibreOffice->Advanced: uncheck "Use a Java runtime environment"

this will make it run faster in the future.

press OK and exit.

in Synaptic search for

kubuntu-restricted-extras

install it.

open terminal.

type

cat /proc/sys/vm/swappiness

press enter. should say 60.

type sudo nano /etc/sysctl.conf

press enter

type in password. you won't see anything while you type, that is how the terminal works. press enter once done

scroll down to end of the file.

write this:

vm.swappiness = 10

press Ctrl+O to save, enter if prompted where to save, then Ctrl+X to exit.

reboot system.

in Synaptic search for

gufw

install it.

launch it. switch it on. close. forget.

open Firefox

Settings->scroll down. Enable Play DRM-controlled content.

tweak firefox as you like.

from synaptic install

qbittorrent

speedcrunch

redshift

plasma-applet-redshift-control

you don't need to install each separately. search for each, mark for installation. after everything is marked then click on Apply. it will install everything then.

after installation

open the applications menu (start menu), type redshift

when the "run redshift" suggestion appears, press enter

you should see a lightbulb in the right side of your taskbar. right click it. Configure Redshift Control

check "Smooth transitions" and "autostart"

click on Advanced

click on locate so that it sets timezone to where you are.

don't fuck with anything else. press Apply, then OK

from synaptic search for

intel-microcode

if already installed leave it, else mark for installation and apply

if your comp is 32 bit then go for i386 version

launch the file manager (dolphin)

from the top panel, click the last icon (three horizontal lines, options). click Configure Toolbars

look for whatever you want on the left, click it, and drag to the right to keep it on your panel. you can also reorder panel items by dragging them up or down or using the buttons between two panels. my right panel looks like:
